FAQ

Apa itu Rekayasa Perangkat Lunak?

Rekayasa Perangkat Lunak (RPL) adalah bidang studi yang mempelajari tentang pengembangan, perancangan, pemeliharaan, dan pengujian perangkat lunak. Di SMKN 20 Jakarta, siswa diajarkan berbagai bahasa pemrograman dan teknik pengembangan software modern.

Apa saja materi yang dipelajari di jurusan RPL?

Materi yang dipelajari meliputi: Pemrograman Dasar, Pemrograman Web (HTML, CSS, JavaScript), Pemrograman Mobile, Database, Sistem Operasi, Jaringan Komputer, dan Proyek Pengembangan Software.

Apa prospek karir lulusan RPL?

Lulusan RPL dapat bekerja sebagai: Web Developer, Mobile Developer, Software Engineer, Database Administrator, System Analyst, UI/UX Designer, dan masih banyak lagi. Industri teknologi saat ini sangat membutuhkan tenaga ahli di bidang ini.
